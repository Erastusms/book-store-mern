
## Tech Stack

**Client:** Vite + ReactJS, TailwindCSS, Axios, Notistack

## Run Locally

Clone the project

```bash
  git clone https://github.com/Erastusms/book-store-mern.git
```

Go to the project directory

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Note first install

```bash
  npm create vite@latest
```
