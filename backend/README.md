
## Tech Stack

**Server:** NodeJS, Express, MongoDB

## Run Locally

Clone the project

```bash
  git clone https://github.com/Erastusms/book-store-mern.git
```

Go to the project directory

```bash
  cd backend
```

Setting config mongodb

```bash
  use mongodb versi 2.2.12 or later
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
